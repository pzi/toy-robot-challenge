# Toy Robot Exercise

![Node.js CI](https://github.com/pzi/toy-robot-challenge/workflows/Node.js%20CI/badge.svg)


## Commands


### `yarn start`

Runs the program with the default input (`input_a.txt`). Use `yarn start ./path_to_file.txt` to use that as input.


### `yarn test`

Runs the test suite.
